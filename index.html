Developing Data Products Course Project

Ven Reddy

Introduction

This is an application that loads Motor Trend's Car Data (mtcars) and allows a person to compare two cars against each other using the UI rather than a command line.

Side panel

The side panel allows you to pick two cars, an attribute to plot. Click on the Submit button to have the application show you the comparison of both cars.

Main panel

The main panel shows you what cars that you have selected and also shows the various attributes for each car such as mpg, cylinders, horsepower, etc.

For example:

# show two cars
data(mtcars)
mtcars$names <- rownames(mtcars)
f <- c("Datsun 710", "Valiant")
filter(mtcars, names %in% f)
